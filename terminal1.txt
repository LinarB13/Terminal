
1. Посмотреть где я - pwd

2. Создать папку - mkdir hw_1

3. Зайти в папку - cd hw_1

4. Создать 3 папки - mkdir h1, mkdir h2, mkdir h3

5. Зайти в любую папку - cd h1

6. Создать 5 файлов (3 txt, 2 json) - touch file1.txt, touch file2.txt, touch file3.txt, touch J1.json, touch J2.json

7. Создать 3 папки - mkdir h11, mkdir h22, mkdir h33

8. Вывести список содержимого папки - ls -la

9. Открыть любой txt файл - cat >> file1.txt

10. Написать туда что-нибудь - qwert (enter) asdf (enter) zxcv (enter) 1234 (enter)

11. Сохранить и выйти - ctrl+c

12. Выйти из папки на уровень выше - cd ..

13. Переместить 2 файла в любую папку - mv h1/file1.txt h2/file1.txt

14. Скопировать любые 2 файла в любую папку - cp h1/J1.json h3/J1.json

15. Найти файл по имени - find . file1.txt

16. Посмотреть содержимое в реальном времени - tile -f file1.txt

17. Вывести несколько первых строк из файла - head -n2 file1.txt

18. Вывести несколько последних строк из файла - tail -n2 file1.txt

19. Посмотреть содержимое длинного файла - less file2.txt
